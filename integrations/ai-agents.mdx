## Introdução

## Como integrar um agente de IA à ZapperAPI

### Pela API

Para realizar a integração de um Agente de IA da GPT Maker ou Zaia na ZapperAPI, consulte a 
<a href="http://docs.zapperapi.com/api-reference/integra%C3%A7%C3%B5es/criar-integra%C3%A7%C3%A3o">Seção Integrações</a> na 
documentação de referência da API para criar ou atualizar os dados de uma integração.

Você deverá passar no corpo da requisição o id do Agente (`agentId`) e o token de API (`apiKey`)
da sua plataforma de agente de IA.

### Pelo Dashboard

A ZapperAPI conta com uma interface amigável para configuração de seu agente de IA:

- Acesse o dashboard em https://app.zapperapi.com/dash;
- Clique sobre a instância desejada;
- Selecione a guia **Integração IA**;
- Preencha os dados de seu agente GPT Maker ou Zaia e clique em **Atualizar Integração**.

## Como controlar o agente pelo WhatsApp

Com a integração no agente de IA, é possível disparar comandos, tanto pela API
quanto pelo próprio dispositivo ou WhatsApp Web, para ativar ou desativar o agente.

- Para **desativar** o agente de IA **via API**, basta mandar uma mensagem de texto 
para o contato com o texto `#off`;
- Para **desativar** o agente de IA **via dispositivo, WhatsApp Web ou Desktop**, 
é possível configurar para desativar mandando **qualquer mensagem de texto**, ou
se preferir, enviando o texto `#off` (como na API);
- Para **reativar** o agente de IA, **tanto pela API quanto pelo dispositivo ou WhatsApp Web/Desktop**,
é só mandar mensagem com texto `#on`.

### Configurando o modo de desativação da IA pelo dispositivo ou web/desktop

Para configurar a forma como a IA é desativada através do dispositivo móvel, WhatsApp
Web ou Desktop:

- Acesse a guia **Integração IA**;
- Marque a opção **Desativar IA ao enviar qualquer mensagem através do dispositivo, WhatsApp Web ou Desktop**.
