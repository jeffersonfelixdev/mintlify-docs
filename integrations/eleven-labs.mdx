## Introdução

**O que é a ElevenLabs**

A ElevenLabs é uma empresa de software especializada no desenvolvimento de aplicações
baseadas em síntese de voz natural, utilizando LLM de aprendizado profundo, com
possibilidade de escolher tom de voz, velocidade, timbre e até sotaques regionais.

O cadastro na ElevenLabs é gratuito e pode ser feito <a href="https://elevenlabs.io">aqui</a>.


## Como integrar a ElevenLabs à ZapperAPI

### Pela API

Para poder utilizar a integração com a ElevenLabs via API, consulte a 
<a href="/api-reference/elevenlabs">Seção ElevenLabs</a> na 
documentação de referência da API para criar ou atualizar a chave de API da ElevenLabs,
bem como escolher uma voz padrão de uso.

### Pelo Dashboard

Para facilitar, disponibilizamos essa configuração pelo Dashboard:

- Acesse o dashboard em https://app.zapperapi.com/dash;
- Clique sobre a instância desejada;
- Selecione a guia **Texto para Fala**;
- Preencha o campo Chave da API ElevenLabs e clique em **Atualizar**.

Você consegue obter uma chave de API da ElevenLabs a partir <a href="https://elevenlabs.io/app/settings/api-keys" target="_blank">deste link</a>.

### Responder mensagens do agente de IA com áudio

Você poderá configurar que seu agente de IA responda mensagens de áudio utilizando
a integração com a ElevenLabs. Para isso, siga os seguintes passos:

- Acesse o dashboard e clique em sua instância;
- Selecione a guia **Configurações**;
- Na seção **Mensagens**, selecione a voz que deseja usar como padrão;
- Na seção **Integração AI**, marque a opção **Responder com áudio**.

Pronto! Sua integração da ElevenLabs agora vai cuidar de todo o processo de 
converter o texto do agente para áudio e enviar para o usuário!
