---
title: Chats
description: "Organize suas conversas pela API"
---

Os chats correspondem às mensagens enviadas para um mesmo número individual ou 
grupo/comunidade. Na ZapperAPI, é possível realizar operações relacionadas aos
chats, como:
- Arquivar ou desarquivar chat
- Silenciar ou reativar notificações de chat
- Fixar ou desfixar chat
- Remover chat

Para maiores informações, consulte a seção *Chat* da API Reference.
