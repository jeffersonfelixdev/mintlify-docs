---
title: Grupos
description: "Gerencie seus grupos e tenha o controle de seus lançamentos"
---

Na ZapperAPI, você poderá realizar as seguintes ações com grupos:

- Criar um grupo ou comunidade
- Exibir detalhes do grupo
- Atualizar informações do grupo, nome e foto
- Gerenciar participantes do grupo, adicionando, removendo, promovendo para admin
ou removendo a permissão admin
- Gerar link de convite do grupo
- Listar grupos que o número participa
- Listar os participantes do grupos

Além de todas as funcionalidades acima, será possível utilizar o módulo de mensagens
para enviar textos e mídias para os grupos.


<Card title="IMPORTANTE">
Caso o número conectado não seja admin do grupo que esteja consultando via API,
é possível que os participantes não tenham o seu número de telefone exibido, em
virtude das restrições de privacidade da Meta. No lugar do número, será exibido
o código LID do contato.
</Card>
